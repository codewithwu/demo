# demo

科研选题副驾子系统 - 选题Agent

## 项目简介

帮助研究者完成科研选题的Agent系统，从研究兴趣、问题域或初步想法出发，经过文献检索、方向全景调研、候选路径比较、收敛判断，最终提供科研选题建议。

## 功能特性

- **意图解析**：将用户模糊想法转为结构化研究问题
- **文献检索**：基于问题检索相关学术论文（Mock数据）
- **方向调研**：梳理研究方向的现状、趋势、空白点
- **路径比较**：对比候选研究路径的可行性、创新性、风险
- **收敛判断**：综合证据评估选题置信度
- **可信度机制**：来源分级（顶会/期刊/预印本/博客）+ 冲突检测


## 核心Workflow

```
用户输入 → 意图解析 → 文献检索 → 方向调研 → 路径比较 → 收敛判断 → 选题建议
   ↓           ↓           ↓           ↓           ↓           ↓
         用户确认    用户确认                 用户确认              用户确认
```

## 来源分级

| 等级 | 来源类型 | 星级 |
|------|----------|------|
| 高 | 顶会/核心期刊 | ★★★ |
| 中 | 普通期刊/会议 | ★★ |
| 低 | 预印本/博客 | ★ |

## 评估维度

- **来源质量**：综合考虑发表载体和引用量
- **证据数量**：支持论文的数量和相关性
- **时效性**：论文年龄的影响（越新权重越高）
- **冲突检测**：识别矛盾结论并提示
